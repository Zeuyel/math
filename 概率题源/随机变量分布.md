# 标准正态

> 看到正态分布的随机变量应该首先想到标准化，其次是泊松积分

设随机变量 $X \sim N(\mu, \sigma^2)(\sigma > 0)$，记 $p = P\{X \le \mu + \sigma^2\}$。

则有：

$$
p = P\left\{   \frac{X - \mu}{\sigma} \le \sigma  \right\} = \Phi(\sigma)
$$

故而 $p$ 只和 $\sigma$ 的 取值有关系

# 条件概率

>1. 题目明确的说明，在B条件下A发生的概率
>2. 离散随机变量通过全概率公式的转换

假设随机变量X的绝对值不大大于1, $P(X=-1) = \frac{1}{8}$ , $P(X=1) = \frac{1}{4}$ , 在事件 $-1<X<$ 1)出现的条件下, X在(-1,1)内的任一子区间上取值的条件概率与该子区间长度成正比.试求X的分布函数.

> 一定要小心处理端点

则：

$$
P\{ -1 < X < x | -1<X< 1 \} = k(x + 1) = \frac{P\{  -1 < X < x \}}{P\{ -1 < X < 1 \}}
$$


得到：

$$
P\{ -1 < X < x \} = k(x+1) P\{ -1 < X < 1 \} = \frac{5}{8} k (x+1)
$$

有一个参数待求，自然使用分布函数的规范性或者概率时间的公理规范性，因此：


$$
P\{  -1 < X < 1 \} = \frac{5}{8} = \frac{5}{8} k (1 +1) \implies k = \frac{1}{2}
$$

因此 

$$F(x) = \frac{1}{8} + \frac{1}{2}(x+1)\cdot\frac{5}{8} = \frac{5x+7}{16}$$

综上所述，

$$F(x) = \begin{cases} 0, & x \le -1, \\ \frac{5x+7}{16}, & -1 < x < 1, \\ 1, & x \ge 1. \end{cases}$$

> 注意分布函数的右连续，区间的左闭右开。

# min-max

设$X$服从指数分布，则$Y=min\{X,2\}$的分布函数( )

一般来说对于随机变量 $Z \triangleq  \min \{ X,Y \}$ ，我们有：

$$
F_{Z}(z) = P \{ X \le z \cup Y \le z \} = P\{ X \le z \} + P\{ Y \le z \} - P\{ X \le Z, Y \le Z \}
$$

如果独立则还有：

$$
F_{Z}(z) = 1 - [1 - P\{ X \le z \}][ 1 - P\{ Y \le z \}]
$$

但是这一题，我们可以直接写出分布函数对应的事件，考虑到离散型随机变量或者常值，我们再进一步使用全概率公式：


$$
F_Y(y) = P(Y \le y) = P(\min\{X,2\} \le y) = P(X \le y, X \le 2) + P(2 \le y, X > 2)
$$

其中 $X \sim E(\lambda)$

当 $y < 0$ 时，$F_Y(y) = p\{X \le y\} = 0$;

当 $0 \le y < 2$ 时，$F_Y(y) = P(X \le y) = \int_0^y \lambda e^{-\lambda x} dx = 1 - e^{-\lambda y}$;

当 $y \ge 2$ 时，$F_Y(y) = P(X \le 2) + P(X > 2) = 1$.

故 
$$F_Y(y) = \begin{cases}
0, & y < 0, \\
1 - e^{-\lambda y}, & 0 \le y < 2, \\
1, & y \ge 2,
\end{cases}$$

它恰有一个间断点。

# 无记忆性

设自动机床在任何时长为 $t$ 的 时间间隔内发生故障的次数 $X$ 服从参数为 $\lambda t$ 的泊松分布， $Y$ 表示相继两次故障之间的时间间隔，则当 $t > 0$ 时，$P\{ Y > t \}$

实际上是泊松过程为背景，重点在于无记忆性，之前发生了多少次故障不会影响之后的故障发生概率。事件 $\{ Y>t \}$ 应当被理解为在 $t$ 间隔当中不发生故障的概率。因此等价于故障发生零次的概率：$e^{-\lambda t}$

# 伯努利试验及  $n$  重伯努利分布

定义：将只有两个结果  $A$  和  $\overline{A}$  的随机试验称为伯努利试验，将伯努利试验独立重复进行  $n$  次称为  $n$  重伯努利试验，记  $n$  重伯努利试验中  $A$  出现的次数为  $X$  ，每次试验中  $A$  出现的概率为  $p$  ，则称  $X$  服从  $n$  重伯努利分布或二项分布，记为  $X\sim B(n,p)$

1）若  $X \sim B(n, p)$ ，当  $k = [(n + 1)p]$  时， $P\{X = k\}$  达到最大值

（换言之，  $X$  最有可能取到  $[(n + 1)p]$  ）

2）若  $X \sim B(n, p)$ ，则当  $p$  充分小而  $n$  充分大且  $np$  适中时， $X$  近似服从参数为  $\lambda = np$  的泊松分布，即  $P\{X = k\} = C_n^k p^k (1 - p)^{n - k} \approx \frac{\lambda^k}{k!} e^{-\lambda}$  （其中  $\lambda = np$ ）（泊松定理）

# 随机变量的无记忆性

几何分布具有“无记忆性”

$$
P (X > m + n | X > m) = P (X > n)
$$

$$
P (X <   m + n | X > m) = P (X <   n)
$$

$$
P (X = m + n \mid X > m) = P (X = n)
$$

指数分布具有“无记忆性”

$$
P (X > s + t | X > t) = P (X > s)
$$

$$
P (X <   s + t | X > t) = P (X <   s)
$$
