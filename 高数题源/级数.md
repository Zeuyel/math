# 级数敛散性判断题型

> 1. 判别法的条件和结论
> 2. 定性分析为主要，把握必要性探路的思想，对常见级数敛散性敏感。
> 3. 把握两个视角：
> 	1. 通项视角
> 		1. 判别法，正项技术判别法，对通项进行分析。
> 		2. 数列研究方法。因当对 [[数列分析]] 在这里也有警惕。比如柯西收敛准则等等，以及最为核心的单调有界性质。结合必要性。
> 	2. 前 $n$ 项和视角
> 		1. 高阶的判别法
> 		2. 两个级数之间的四则运算
> 			1. 加减用性质
> 			2. 乘积可以考虑A-D判别法。
> 			3. 高中的数列求和，比如特征明显的裂项等。
> 4. 由于定性分析，抽象题目可以举出例子来把握住最终发挥作用的部分。

下面都是关于通项的，一方面考虑判别法，另一方面研究其数列性质。但是可惜的是，这里没有给出正项级数的条件，同时没有绝对收敛等转换。

回归到极限，$\lim_{ n \to \infty } a_{n} b_{n} = 0$ ，只能给出 $a_{n}b_{n}$ 在无穷远处的状态，而无法分开说明。或者换一种说法，要使得 下式成立

$$\lim_{ n \to \infty } a_{n}b_{n} = \lim_{ n \to \infty } a_{n} \times \lim_{ n \to \infty }b_{n}$$

需要等式后面的极限都存在。紧接，还需要说明极限不存在不一定意味着是广义极限（$\pm \infty$ 正如名字他们在广义上是可以被接纳为极限的，因为起码保持了在无穷远处的趋势）。故而需要指出的是震荡数列或者函数才是最典型的极限不存在，比如：

$$
\begin{align}
a_{n} = 0,1,0,1\dots  \\
b_{n} = 1,0,1,0\dots
\end{align}
$$

> 极限的反例，往往应当尝试震荡，最后得到的常数列。

因此这个条件是比较弱的，我们应当思考通过必要性证否。

(A) 若$\lim_{n \to \infty} a_nb_n = 0$，则$\sum_{n=1}^{\infty} a_n$ 和 $\sum_{n=1}^{\infty} b_n$ 中至少有一个收敛。

(B) 若$\lim_{n \to \infty} a_nb_n = 1$，则$\sum_{n=1}^{\infty} a_n$ 和 $\sum_{n=1}^{\infty} b_n$ 中至少有一个发散。

而对于 $\lim_{ n \to \infty } a_{n}b_{n} = 1$  我们则可以推出， $\lim_{ n \to \infty }a_{n} = 0$ 和 $\lim_{ n \to \infty }b_{n} = 0$ 不可能同时成立。故而由级数收敛的必要性得到，至少有一个发散。

极限形式的比值判别法，仅适用于正项级数，无法对一般级数进行归纳：

(C) 若$\lim_{n \to \infty} \frac{a_n}{b_n} = 0$，则$\sum_{n=1}^{\infty} a_n$ 收敛时 $\sum_{n=1}^{\infty} b_n$ 收敛。

(D) 若$\lim_{n \to \infty} \frac{a_n}{b_n} = \infty$，则$\sum_{n=1}^{\infty} b_n$ 发散时 $\sum_{n=1}^{\infty} a_n$ 发散。

# 幂级数收敛域题型

**标准步骤**：
1. 求收敛半径（比值法或根值法）
2. 写出收敛区间
3. **单独检验端点**（代入具体值判断）

> [!warning] 常见错误
> - 忘记检验端点
> - 缺项级数直接用公式（应先换元或用根值法）

# 幂级数求和

**判断方法选择**：
1. 通项分母含 $n$ → 先求导后积分
2. 通项分子含 $n$ → 先积分后求导
3. 通项含 $(n+1)(n+2)$ 等 → 拆分后分别处理

# 函数展开题型

**方法选择**：
1. 能用已知展开式变形的优先间接法
2. 有理函数 → 部分分式分解后展开
3. 复杂函数 → 考虑求导/积分转化

# 七、重要结论与公式速查

## 1. 敛散性快速判断

| 级数类型                                 | 收敛条件        |
| ------------------------------------ | ----------- |
| $\sum q^n$                           | $\|q\| < 1$ |
| $\sum \frac{1}{n^p}$                 | $p > 1$     |
| $\sum \frac{1}{n(\ln n)^p}$          | $p > 1$     |
| $\sum \frac{1}{n\ln n(\ln\ln n)^p}$  | $p > 1$     |
| $\sum (-1)^n u_n$（$u_n\downarrow 0$） | 收敛          |

## 2. 常用求和公式

$$\sum_{n=0}^{\infty}x^n = \frac{1}{1-x}, \quad |x|<1$$

$$\sum_{n=1}^{\infty}nx^{n-1} = \frac{1}{(1-x)^2}, \quad |x|<1$$

$$\sum_{n=1}^{\infty}nx^n = \frac{x}{(1-x)^2}, \quad |x|<1$$

$$\sum_{n=1}^{\infty}\frac{x^n}{n} = -\ln(1-x), \quad -1\leq x<1$$

$$\sum_{n=0}^{\infty}\frac{x^{2n+1}}{2n+1} = \frac{1}{2}\ln\frac{1+x}{1-x}, \quad |x|<1$$

## 3. 运算性质

- **逐项求导**：收敛半径不变，端点需重新判断
- **逐项积分**：收敛半径不变，端点需重新判断
- **级数乘法**（柯西乘积）：$\left(\sum a_n\right)\left(\sum b_n\right) = \sum c_n$，其中 $c_n = \sum_{k=0}^{n}a_kb_{n-k}$

## 例题1：级数敛散性综合判断

**题目**：判断级数 $\sum_{n=1}^{\infty}\frac{(-1)^n}{\sqrt{n}+(-1)^n}$ 的敛散性。

**解**：
$$u_n = \frac{(-1)^n}{\sqrt{n}+(-1)^n} = \frac{(-1)^n(\sqrt{n}-(-1)^n)}{n-1}$$

当 $n$ 为偶数：$u_n = \frac{\sqrt{n}-1}{n-1} \sim \frac{1}{\sqrt{n}}$

当 $n$ 为奇数：$u_n = \frac{-\sqrt{n}-1}{n-1} \sim -\frac{1}{\sqrt{n}}$

故 $\sum|u_n| \sim \sum\frac{1}{\sqrt{n}}$ 发散，原级数不绝对收敛。

改写：$u_n = \frac{(-1)^n}{\sqrt{n}}\cdot\frac{1}{1+\frac{(-1)^n}{\sqrt{n}}}$

由于 $\frac{(-1)^n}{\sqrt{n}}$ 满足莱布尼兹条件，且 $\frac{1}{1+\frac{(-1)^n}{\sqrt{n}}} \to 1$，可证原级数**条件收敛**。

## 例题2：收敛域求解

**题目**：求 $\sum_{n=1}^{\infty}\frac{3^n+(-2)^n}{n}(x-1)^n$ 的收敛域。

**解**：
$$\left|\frac{u_{n+1}}{u_n}\right| = \left|\frac{3^{n+1}+(-2)^{n+1}}{3^n+(-2)^n}\right|\cdot\frac{n}{n+1}\cdot|x-1|$$

$$= \frac{3+(-2)\cdot(-\frac{2}{3})^n}{1+(-\frac{2}{3})^n}\cdot\frac{n}{n+1}\cdot|x-1| \to 3|x-1|$$

故 $R = \frac{1}{3}$，收敛区间 $\left(\frac{2}{3}, \frac{4}{3}\right)$

端点检验：
- $x = \frac{4}{3}$：$\sum\frac{3^n+(-2)^n}{n\cdot 3^n} = \sum\frac{1}{n} + \sum\frac{(-\frac{2}{3})^n}{n}$，发散
- $x = \frac{2}{3}$：$\sum\frac{(-1)^n(3^n+(-2)^n)}{n\cdot 3^n}$，收敛（类似交错级数）

**收敛域**：$\left[\frac{2}{3}, \frac{4}{3}\right)$

## 例题3：幂级数求和

**题目**：求 $\sum_{n=1}^{\infty}\frac{x^n}{n(n+1)}$ 的和函数。

**解**：收敛域为 $[-1, 1]$

拆分：$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$

$$S(x) = \sum_{n=1}^{\infty}\frac{x^n}{n} - \sum_{n=1}^{\infty}\frac{x^n}{n+1}$$

$$= -\ln(1-x) - \frac{1}{x}\sum_{n=1}^{\infty}\frac{x^{n+1}}{n+1}$$

$$= -\ln(1-x) - \frac{1}{x}\left[-\ln(1-x) - x\right]$$

$$= -\ln(1-x) + \frac{\ln(1-x)}{x} + 1$$

$$= 1 + \frac{(1-x)\ln(1-x) - \ln(1-x)}{x} = 1 + \frac{x-1}{x}\ln(1-x) + \frac{\ln(1-x)}{x}$$

整理得：$S(x) = 1 + \left(1 - \frac{1}{x}\right)\ln(1-x)$（$x \neq 0$）

当 $x = 0$：$S(0) = 0$

## 例题4：函数展开成幂级数

**题目**：将 $f(x) = x\arctan x - \ln\sqrt{1+x^2}$ 展开成 $x$ 的幂级数。

**解**：
$$\arctan x = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n+1}}{2n+1}$$

$$x\arctan x = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n+2}}{2n+1}$$

$$\ln(1+x^2) = \sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$\ln\sqrt{1+x^2} = \frac{1}{2}\sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$f(x) = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n+2}}{2n+1} - \frac{1}{2}\sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$= \sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{2n-1} + \frac{1}{2}\sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$= \sum_{n=1}^{\infty}(-1)^{n-1}\left(\frac{1}{2n-1} + \frac{1}{2n}\right)\frac{x^{2n}}{2}$$

$$= \sum_{n=1}^{\infty}(-1)^{n-1}\frac{4n-1}{2n(2n-1)}x^{2n}, \quad |x| \leq 1$$
