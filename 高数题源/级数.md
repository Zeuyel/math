# 级数概念

> 1. 定性分析为主要，把握必要性探路的思想，对常见的（广义）$p$级数敛散性敏感。
> 	1. 通项收敛到 $0$ 的必要条件
> 	2. 反证法与逆否命题是这里的主要技术细节
> 2. 掌握两个视角：
> 	1. 通项视角
> 		1. 判别法，正项极限判别法+ 莱布尼兹交错判别法，对通项进行分析。
> 			1. 此时我们的工具库扩展到极限研究的相应范式。
> 		2. 数列研究方法。应当当对 [[数列分析]] 在这里也有警惕。比如柯西收敛准则等等，以及最为核心的单调有界性质。再结合必要性。
> 	2. 前 $n$ 项和视角
> 		1. 两个级数之间的四则运算
> 			1. 加减用性质（本质是极限的四则运算）
> 			2. 乘积可以考虑A-D判别法，或者转化为 $p$ 级数
> 			3. 数列求和，比如特征明显的裂项等（类似于强行裂项，应当在幂级数求和等考察）。
> 	3. 显然的，前 $n$ 项和得出的渐近估计强于通项。但是，往往我们思考的时候只能在通项上思考，这也是可以的，我们可以在直觉上把握通项充足的小，比如远小于 $\frac{1}{n^{2}}$ 或者 $\frac{1}{n} - \frac{1}{n+1}$ 然后在前 $n$ 项和上说明。
> 3. 抓住两个层次
> 	1. 绝对收敛
> 	2. 条件收敛
> 4. 由于定性分析，抽象题目可以举出例子来把握住最终发挥作用的部分。
> 	1. 如果命题对于正向级数正确，则考虑交错级数作为反例
> 	2. 常数列
> 	3. 增加条件为构造反例指引方向


## 简单通项分析

> 我们指出，级数审敛判别的细节非常多，故而造成简单考点容易遗忘，或者条件容易缺失。
> 对于常用的审敛法则，关键在于把握住首先对级数分类把握，不要混淆我们的正项、交错、一般项级数
> 另一方面，使用好极限这个工具，他能够描述出无穷远处通项的性状。

### 例题1

对无穷级数 $\sum_{n=1}^{\infty} a_n$ 和 $\sum_{n=1}^{\infty} b_n$，下列命题成立的是( )

下面都是关于通项的，一方面考虑判别法，另一方面研究其数列性质。但是可惜的是，这里没有给出正项级数的条件，同时没有绝对收敛等转换。

回归到极限，$\lim_{ n \to \infty } a_{n} b_{n} = 0$ ，只能给出 $a_{n}b_{n}$ 在无穷远处的状态，而无法分开说明。或者换一种说法，要使得 下式成立

$$\lim_{ n \to \infty } a_{n}b_{n} = \lim_{ n \to \infty } a_{n} \times \lim_{ n \to \infty }b_{n}$$

需要等式后面的极限都存在。紧接，还需要说明极限不存在不一定意味着是广义极限（$\pm \infty$ 正如名字他们在广义上是可以被接纳为极限的，因为起码保持了在无穷远处的趋势）。故而需要指出的是震荡数列或者函数才是最典型的极限不存在，比如：

$$
\begin{align}
a_{n} = 0,1,0,1\dots  \\
b_{n} = 1,0,1,0\dots
\end{align}
$$

> 极限的反例，往往应当尝试分开为震荡，合并为常数列。

因此这个条件是比较弱的，我们应当思考通过必要性证否。

(A) 若$\lim_{n \to \infty} a_nb_n = 0$，则$\sum_{n=1}^{\infty} a_n$ 和 $\sum_{n=1}^{\infty} b_n$ 中至少有一个收敛。

(B) 若$\lim_{n \to \infty} a_nb_n = 1$，则$\sum_{n=1}^{\infty} a_n$ 和 $\sum_{n=1}^{\infty} b_n$ 中至少有一个发散。

而对于 $\lim_{ n \to \infty } a_{n}b_{n} = 1$  我们则可以推出， $\lim_{ n \to \infty }a_{n} = 0$ 和 $\lim_{ n \to \infty }b_{n} = 0$ 不可能同时成立。故而由级数收敛的必要性得到，至少有一个发散。

极限形式的比值判别法，仅适用于正项级数，无法对一般级数进行归纳：

(C) 若$\lim_{n \to \infty} \frac{a_n}{b_n} = 0$，则$\sum_{n=1}^{\infty} a_n$ 收敛时 $\sum_{n=1}^{\infty} b_n$ 收敛。

(D) 若$\lim_{n \to \infty} \frac{a_n}{b_n} = \infty$，则$\sum_{n=1}^{\infty} b_n$ 发散时 $\sum_{n=1}^{\infty} a_n$ 发散。

### 例题1

(A) $\sum_{n=1}^{\infty} (a_n+a_{n+1})$收敛.

看成 $\sum a_{n}$ 和 $\sum a_{n+1}$ 两个级数的和，显然两个级数都收敛，由性质收敛加收敛仍然收敛。

(B) $\sum_{n=1}^{\infty} (a_n^2-a^{2}_{n+1})$收敛.

经典裂项结构，直接求和：

$$
\lim_{ n \to \infty } \sum _{n=1}^{\infty} (a_{n}^{2} - a_{n+1}^{2}) = a_{1}^{2} - \lim_{ n \to \infty } a_{n}^{2} = a_{1}^{2}
$$

(C) $\sum_{n=1}^{\infty} (a_{2n}+a_{2n+1})$收敛.

这相当于是 $\left\{  \sum _{n=0}^{\infty} a_{n}  \right\}$ 的偶子列

(D) $\sum_{n=1}^{\infty} (a_{2n}-a_{2n+1})$收敛.

这拆分开是缺项数列，只有绝对收敛的数列才能保证分开的缺项仍然是收敛的。


### 例题2

若 $\sum _{n=1}^{\infty} u_{n}$ 和 $\sum _{n=1}^{\infty} v_{n}$ 收敛，证明 $\sum _{n=1}^{\infty} (u_{n}\pm v_{n})$ 收敛

最初始的思路是根据显然的通项关系证明，然而，没有正项的条件。同时又缺少阶的信息，这是难以为继。故而我们只能转换到最本质的：前$n$ 项和极限存在。

此时要转换一下：

$$
c_{n} = \sum _{i=1}^{n}(u_{i} \pm v_{i}) = \sum _{i= 1}^{n}u_{i} \pm \sum _{i}^{n}v_{i} = S_{n} + T_{n}
$$

> 转化为有限项求和拆分求和号。

故而：

$$
\lim_{ n \to \infty } c_{n} = \lim_{ n \to \infty } S_{n} \pm \lim_{ n \to \infty } T_{n} = A\pm B \implies \sum _{n = 1} ^{\infty} (u_{i}\pm v_{i}) = \sum _{n=1}^{\infty} u_{i} \pm \sum _{n=1}^{\infty} v_{i}
$$

故而我们还能够得到：

- 收敛+发散 = 发散
- 注意：发散+发散 的结果将不一定，因为两个都不确定在无穷远处的性状。

# 幂级数求和

> [!abstract] 级数求和
> 1. 数项级数求和含于幂级数求和当中，相当于函数表达式和函数值的关系。而收敛域则是广义的函数定义域，因此对于数项级数，只需要保证要求点有意义即可。
> 2. 幂级数求和，我们分为三个类型
> 	1. 整式级数
> 	2. 分式级数
> 	3. 阶乘级数
> 	4. 不仅要记忆题型，而是把握住我们的“佩奇”思想，和把 $n$ 和 $x$ 分开处理的基本逻辑（$n$ 是随着通项不同而变动的，而$x$ 则是一致的）。
> 3. 要把握好$a_{n}$ 和 $n$ 之间的相对关系，由于 $n$ 最终趋向于无穷，故而有一些筛法本质上相同的。
> 4. $0^{0} =1$
> 5. 这里的基础是[[Taylor展开与麦克劳林级数]]，一方面对于我们下面说的所有常见级数展开要实现“倒背如流”，另一方面，我们也给出一种从朴素的二项式定理的各个级数展开式的推导方式。

## 收敛域

1. 求收敛半径（比值法或根值法）
2. 写出收敛区间
3. **单独检验端点**

> [!warning] 常见错误
> - 忘记检验端点
> - 缺项级数应先换元或用根值法

### 例题

求 $\sum_{n=1}^{\infty}\frac{3^n+(-2)^n}{n}(x-1)^n$ 的收敛域。

$$
\begin{align}
\lim_{ n \to \infty } \left|\frac{u_{n+1}}{u_n}\right|  & = \lim_{ n \to \infty } \left|\frac{3^{n+1}+(-2)^{n+1}}{3^n+(-2)^n}\right|\cdot\frac{n}{n+1}\cdot|x-1|  \\
 & = \lim_{ n \to \infty }  \frac{3+(-2)\cdot(-\frac{2}{3})^n}{1+(-\frac{2}{3})^n}\cdot\frac{n}{n+1}\cdot|x-1|  \\
 & = 3|x-1|
\end{align}
$$


故 $R = \frac{1}{3}$，收敛区间 $\left(\frac{2}{3}, \frac{4}{3}\right)$

端点检验：
- $x = \frac{4}{3}$：$\sum\frac{3^n+(-2)^n}{n\cdot 3^n} = \sum\frac{1}{n} + \sum\frac{(-\frac{2}{3})^n}{n}$，发散
- $x = \frac{2}{3}$：$\sum\frac{(-1)^n(3^n+(-2)^n)}{n\cdot 3^n}$

下证 $a_{n} \triangleq \sum _{n=1}^{\infty} \frac{(-1)^{n}(3^{n} + (-2)^{n})}{n 3^{n}}$ 收敛

$$
\sum_{n=1}^{\infty} a_n = \sum_{n=1}^{\infty} \frac{(-1)^n}{n} + \frac{2^n}{n^3} = \sum_{n=1}^{\infty} \frac{(-1)^n}{n} + \sum_{n=1}^{\infty} \frac{1}{n}(\frac{2}{3})^n
$$

前一个是交错调和级数，满足莱布尼兹准则

后一个我们使用正项极限判别法：

$$
\lim_{ n \to \infty } n^{p-1} \left( \frac{2}{3} \right) ^{n} = \lim_{ n \to \infty } \frac{n^{p-1}}{\left( \frac{3}{2} \right) ^{n}} 
$$

显然幂级数恒弱于指数函数，不妨取 $p = 2$ 故而收敛。当然这里有指数，也可以使用根值判别法。

综上，得到收敛域：$\left[\frac{2}{3}, \frac{4}{3}\right)$

### 例题1

求级数 $\sum_{n=1}^{\infty} \frac{1}{n(n+1)} x^n$ 的和函数

首先计算收敛半径：

$$
R = \lim_{ n \to \infty } \frac{n(n+1)}{(n+2)(n+3)} = 1
$$

> 由于后续我们的化简操作中可能有其他需要独立讨论的点，所有收敛半径在最后说明。

这里和积分一样，我们几乎无法处理高次的$n$ ，请务必不要遗漏我们在积分中处理这部分分式的技巧。首先分项得到

$$
s(x) = \sum _{n=1}^{\infty} \frac{1}{n} x^{n} - \sum _{n=1}^{\infty} \frac{1}{n+1}x^{n} = I_{1} - I_{2}
$$

那么我们有：

$$
\begin{align}
I_{1}  & = -\ln (1-x), x\in [-1,1) \\
I_{2}  & = \frac{1}{x} \sum _{n=1}^{\infty} \frac{1}{n+1} x^{n+1} = \frac{1}{x} \sum _{n=2}^{\infty} \frac{1}{n} x^{n} = \frac{1}{x} \left[ -\ln(1-x) - x \right] , x \in [-1,1) \\
s(x)  & =I_{1} - I_{2} =  \left( \frac{1}{x} - 1 \right) \ln(1-x) + 1
\end{align}
$$

现在讨论剩下的两个点：

- $x = 0$ 这个点可以直接代入，原式等于 $\sum _{n=0}^{\infty} 0 = 0$ 
- $x=1$ 这个点可以直接代入然后裂项说明极限存在。也可以对上式取极限：

当 $x=1$ 时：

$$
s(x)= \lim_{ x \to 1^{-} } I_{1} - I_{2} = \lim_{ x \to 1^{-} }  \frac{(1-x)\ln(1-x)}{x} + 1 = 1
$$

综上：

$$
s(x) = \begin{cases}
\left( \frac{1}{x} -1 \right) \ln(1-x) + 1  & -1 \le x < 1 , x\neq 0 \\
0 & x=0 \\
1 & x=1
\end{cases}
$$

### 构建微分方程

利用幂级数的逐项求导性质，可以建立和函数满足的微分方程。

**基本思路**：
1. 设和函数 $S(x) = \sum_{n=0}^{\infty}a_nx^n$
2. 求 $S'(x)$、$S''(x)$ 等
3. 寻找 $S(x)$、$S'(x)$、$xS(x)$ 等之间的关系
4. 建立微分方程并求解

> [!example] 典型例题
> 求 $\sum_{n=0}^{\infty}\frac{x^{2n+1}}{(2n+1)!}$ 的和函数
>
> **解**：设 $S(x) = \sum_{n=0}^{\infty}\frac{x^{2n+1}}{(2n+1)!}$
>
> $$S'(x) = \sum_{n=0}^{\infty}\frac{x^{2n}}{(2n)!}$$
>
> $$S''(x) = \sum_{n=1}^{\infty}\frac{x^{2n-1}}{(2n-1)!} = \sum_{n=0}^{\infty}\frac{x^{2n+1}}{(2n+1)!} = S(x)$$
>
> 得微分方程：$S'' - S = 0$，通解为 $S = C_1e^x + C_2e^{-x}$
>
> 初值条件：$S(0) = 0$，$S'(0) = 1$
>
> 解得：$C_1 = \frac{1}{2}$，$C_2 = -\frac{1}{2}$
>
> **和函数**：$S(x) = \frac{e^x - e^{-x}}{2} = \sinh x$

| 级数类型                                 | 收敛条件        |

## 例题3：幂级数求和

**题目**：求 $\sum_{n=1}^{\infty}\frac{x^n}{n(n+1)}$ 的和函数。

**解**：收敛域为 $[-1, 1]$

拆分：$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$

$$S(x) = \sum_{n=1}^{\infty}\frac{x^n}{n} - \sum_{n=1}^{\infty}\frac{x^n}{n+1}$$

$$= -\ln(1-x) - \frac{1}{x}\sum_{n=1}^{\infty}\frac{x^{n+1}}{n+1}$$

$$= -\ln(1-x) - \frac{1}{x}\left[-\ln(1-x) - x\right]$$

$$= -\ln(1-x) + \frac{\ln(1-x)}{x} + 1$$

$$= 1 + \frac{(1-x)\ln(1-x) - \ln(1-x)}{x} = 1 + \frac{x-1}{x}\ln(1-x) + \frac{\ln(1-x)}{x}$$

整理得：$S(x) = 1 + \left(1 - \frac{1}{x}\right)\ln(1-x)$（$x \neq 0$）

当 $x = 0$：$S(0) = 0$

## 例题4：函数展开成幂级数

**题目**：将 $f(x) = x\arctan x - \ln\sqrt{1+x^2}$ 展开成 $x$ 的幂级数。

**解**：
$$\arctan x = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n+1}}{2n+1}$$

$$x\arctan x = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n+2}}{2n+1}$$

$$\ln(1+x^2) = \sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$\ln\sqrt{1+x^2} = \frac{1}{2}\sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$f(x) = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n+2}}{2n+1} - \frac{1}{2}\sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$= \sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{2n-1} + \frac{1}{2}\sum_{n=1}^{\infty}(-1)^{n-1}\frac{x^{2n}}{n}$$

$$= \sum_{n=1}^{\infty}(-1)^{n-1}\left(\frac{1}{2n-1} + \frac{1}{2n}\right)\frac{x^{2n}}{2}$$

$$= \sum_{n=1}^{\infty}(-1)^{n-1}\frac{4n-1}{2n(2n-1)}x^{2n}, \quad |x| \leq 1$$
